#Dupee

![alt text](https://travis-ci.org/walidsa3d/dupee.svg?branch=v0.1 "Build")
A duplicate file finder
##Overview

##Install (automatic)
```
$ pip install dupee
```
##Install (manual)
```
$ git clone git@github.com:walidsa3d/dupee.git
$ cd dupee
$ python setup.py install
```
##Usage
```
$ dupee [options] <rootdir>
```
